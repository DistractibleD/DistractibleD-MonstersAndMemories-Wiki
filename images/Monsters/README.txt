Monster screenshots go in this folder, named to match the "image" path
listed for that monster in monsters.json (at the root of the project).

Screenshots here are saved as .jpg at quality 90, not .png — same
reasoning as item/recipe screenshots, see CLAUDE.md ("Item screenshot
format").

To add a new monster to the Monsters page, add an entry to monsters.json
and drop its picture here with a matching filename. No other files need
to change — the map filter and sort columns pick up new monsters
automatically.
