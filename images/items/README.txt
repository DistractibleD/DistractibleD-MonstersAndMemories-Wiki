Item screenshots go in this folder, named to match the "image" path
listed for that item in items.json (at the root of the project).

Screenshots here are saved as .jpg at quality 90, not .png — see
CLAUDE.md ("Item screenshot format") for why.

To add a new item to the Item Database page, add an entry to items.json
and drop its screenshot here with a matching filename. No other files
need to change — the filters, search box, and table pick up new items
automatically.
