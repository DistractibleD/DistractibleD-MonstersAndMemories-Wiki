Items with possibly missing information
=========================================

This file tracks items where the *data* in items.json might be incomplete — not whether
the saved screenshot's border/crop is complete. Since 2026-07-04 the site builds its own
card from items.json and no longer displays the screenshot at all, so a screenshot that's
visually cut off (missing border, text running to the raw edge) no longer matters on its
own — only whether the actual stats/text made it into items.json. Screenshots are still
kept (as .jpg) purely as the reference used to extract that data, per the user's
2026-07-04 confirmation.
